# Используйте метод карты для извлечения данных из массива
До сих пор мы научились использовать чистые функции, чтобы избежать побочных эффектов в программе. Кроме того, мы увидели ценность функции, зависящей только от ее входных аргументов.

Это только начало. Как следует из названия, функциональное программирование сосредоточено вокруг теории функций.

Было бы разумно иметь возможность передавать их в качестве аргументов другим функциям и возвращать функцию из другой функции. Функции считаются объектами первого класса в JavaScript, что означает, что их можно использовать как любой другой объект. Их можно сохранять в переменных, хранить в объекте или передавать в качестве аргументов функции.

Начнем с некоторых простых функций массива, которые являются методами прототипа объекта массива. В этом упражнении мы рассматриваем `Array.prototype.map()`, или проще говоря `map`.

Метод `map`перебирает каждый элемент массива и возвращает новый массив, содержащий результаты вызова функции обратного вызова для каждого элемента. Он делает это без изменения исходного массива.

Когда используется обратный вызов, ему передаются три аргумента. Первый аргумент — текущий обрабатываемый элемент. Второй — это индекс этого элемента, а третий — массив, для которого `map`был вызван метод.

Ниже приведен пример использования `map`метода для `users`массива для возврата нового массива, содержащего только имена пользователей в качестве элементов. Для простоты в примере используется только первый аргумент обратного вызова.
```javascript
const users = [
  { name: 'John', age: 34 },
  { name: 'Amy', age: 20 },
  { name: 'camperCat', age: 10 }
];

const names = users.map(user => user.name);
console.log(names);
```
Консоль отобразит значение `[ 'John', 'Amy', 'camperCat' ]`.

Массив `watchList`содержит объекты с информацией о нескольких фильмах. Используйте `mapon watchList`, чтобы присвоить `ratings`переменной новый массив объектов. Каждый фильм в новом массиве должен иметь только `title`ключ с названием фильма и `rating`ключ с рейтингом IMDB. Код в редакторе в настоящее время использует `for`для этого цикл, поэтому вам следует заменить функцию цикла своим `map`выражением.
