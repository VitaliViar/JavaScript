Используйте метод сокращения для анализа данных
Array.prototype.reduce(), или просто reduce(), является наиболее общей из всех операций с массивами в JavaScript. reduceС помощью метода можно решить практически любую задачу обработки массива .

Этот reduceметод допускает более общие формы обработки массивов, и можно показать, что и то, filterи другое mapможет быть получено как специальное приложение reduce. Метод reduceперебирает каждый элемент массива и возвращает одно значение (т.е. строку, число, объект, массив). Это достигается с помощью функции обратного вызова, которая вызывается на каждой итерации.

Функция обратного вызова принимает четыре аргумента. Первый аргумент известен как аккумулятор, которому присваивается возвращаемое значение функции обратного вызова из предыдущей итерации, второй — текущий обрабатываемый элемент, третий — индекс этого элемента, а четвертый — массив, на reduceкотором называется.

В дополнение к функции обратного вызова, reduceимеет дополнительный параметр, который принимает начальное значение для аккумулятора. Если этот второй параметр не используется, то первая итерация пропускается, а второй итерации передается первый элемент массива в качестве аккумулятора.

Ниже приведен пример использования reduceмассива usersдля возврата суммы возрастов всех пользователей. Для простоты в примере используются только первый и второй аргументы.

const users = [
  { name: 'John', age: 34 },
  { name: 'Amy', age: 20 },
  { name: 'camperCat', age: 10 }
];

const sumOfAges = users.reduce((sum, user) => sum + user.age, 0);
console.log(sumOfAges);
Консоль отобразит значение 64.

В другом примере посмотрите, как можно вернуть объект, содержащий имена пользователей в качестве свойств с их возрастом в качестве значений.

const users = [
  { name: 'John', age: 34 },
  { name: 'Amy', age: 20 },
  { name: 'camperCat', age: 10 }
];

const usersObj = users.reduce((obj, user) => {
  obj[user.name] = user.age;
  return obj;
}, {});
console.log(usersObj);
Консоль отобразит значение { John: 34, Amy: 20, camperCat: 10 }.
